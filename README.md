## 📌 Project Overview

This project demonstrates a Microsoft 365 tenant auditing workflow capable of:

- Enumerating tenant users
- Exporting audit-ready CSV reports
- Generating HTML reporting dashboards
- Connecting securely to Microsoft Graph
- Using Exchange Online PowerShell for administration
- Producing enterprise-style reporting outputs

The toolkit simulates real-world Microsoft 365 administration and operational reporting workflows commonly used in enterprise IT environments.
